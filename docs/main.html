Passive System Theory and Passivity-Based Control (PBC)

Dissipativity

A dynamical system

Σ:{ẋ=f(x,u)y=h(x,u)x∈𝒳⊂ℝ2n,u∈𝒰⊂ℝm \Sigma: \quad \begin{cases} \dot{x} &= f(x,u) \\ y &= h(x,u) \end{cases} \quad x \in \mathcal{X} \subset \mathbb{R}^{2n}, \, u \in \mathcal{U} \subset \mathbb{R}^{m}

is dissipative with respect to some supply rate ss if there exists a storage function ℋ:𝒳→ℝ+\mathcal{H}: \mathcal{X} \to \mathbb{R}^{+} such that

ℋ(x(t1))≤ℋ(x(t0))+∫t0t1s(u(t),y(t))dt \mathcal{H}\left( x(t_1) \right) \leq \mathcal{H}\left( x(t_0) \right) + \int_{t_0}^{t_1} s(u(t), y(t)) \, \text{d}t

for all x(t0)=x0x(t_0) = x_0, all input uu, and all t1≥t0t_1 \geq t_0

Dissipativity

ℋ(x(t1))≤ℋ(x(t0))+∫t0t1s(u(t),y(t))dt \mathcal{H}\left( x(t_1) \right) \leq \mathcal{H}\left( x(t_0) \right) + \int_{t_0}^{t_1} s(u(t), y(t)) \, \text{d}t

Dissipation Inequality

  • Stored energy at t1t_1 is at most equal to stored energy at t0t_0, plus externally supplied energy s(u,y)s(u,y)
  • No generation of energy, only internal dissipation
  • With s(t)≡0s(t) \equiv 0, trajectories tend towards minimum of ℋ\mathcal{H}

Passivity

The system Σ\Sigma is passive if it is dissipative with supply rate

s=u⊤y.s = u^\top y.

It is output strictly passive if it is dissipative with supply rate

s=u⊤y−δ∥y∥2,δ>0.s = u^\top y - \delta \lVert y \rVert^2, \; \delta > 0.

It is input strictly passive if it is dissipative with supply rate

s=u⊤y−δ∥u∥2,δ>0.s = u^\top y - \delta \lVert u \rVert^2, \; \delta > 0.

Kirchoff’s law

v=Ri+1C∫0ti(τ)dτ+Ldidtvi−Ri2=ddt(12C(∫0ti(τ)dτ)2⏟𝒱+12Li2⏟𝒯) \begin{aligned} v &= Ri + \frac{1}{C} \int_{0}^{t} i(\tau) \, \text{d}\tau + L \frac{\text{d} i}{\text{d} t} \\ vi - Ri^2 &= \frac{\text{d}}{\text{d} t} \left( \underbrace{\frac{1}{2C} \left( \int_{0}^{t} i(\tau)\, \text{d} \tau \right)^2}_{\mathcal{V}} + \underbrace{\frac{1}{2} Li^2}_{\mathcal{T}} \right) \end{aligned}

Let ℋ=𝒱+𝒯\mathcal{H} = \mathcal{V} + \mathcal{T}, integrate to obtain

ℋ(t)⏟available−ℋ(0)⏟initial=∫0tv(τ)i(τ)dτ⏟supplied−∫0tRi2(τ)dτ⏟dissipated<∫0tv(τ)i(τ)dτ \underbrace{\mathcal{H}(t)}_{\textrm{available}} - \underbrace{\mathcal{H}(0)}_{\textrm{initial}} = \underbrace{\int_{0}^{t} v(\tau)i(\tau)\, \text{d} \tau}_{\textrm{supplied}} - \underbrace{\int_{0}^{t} Ri^2(\tau)\, \text{d} \tau}_{\textrm{dissipated}} < \int_{0}^{t} v(\tau) i(\tau) \, \text{d} \tau

Stability of Passive Systems

Σ:{ẋ=f(x,u),f(0,0)=0,y=h(x,u),h(0,0)=0, \Sigma: \quad \begin{cases} \dot{x} &= f(x,u), && f(0,0) = 0, \\ y &= h(x,u), && h(0,0) = 0, \end{cases}

u⊤y≥ℋ̇=∂ℋ∂xf(x,u),ℋ≥0,y≡0⟹x≡0u^{\top} y \geq \dot{\mathcal{H}} = \frac{\partial\mathcal{H}}{\partial x} f(x,u),\quad \mathcal{H} \geq 0,\quad y \equiv 0 \implies x \equiv 0

Lemma (Khalil, 2002)

The origin of Σ\Sigma is:

  • stable if Σ\Sigma is passive,
  • asymptotically stable if Σ\Sigma is output strictly passive,
  • globally asymptotically stable if Σ\Sigma is output strictly passive and the storage function ℋ(x)→∞\mathcal{H}(x) \to \infty as ∥x∥→∞\lVert x \rVert \to \infty (radially unbounded)
  • Notice that we only require that the function ℋ\mathcal{H} be positive semidefinite and not necessarily positive definite as per usual Lyapunov conditions.
  • Observability (detectability) ensures LaSalle’s theorem works.

Σo:{ẋ=f(x)+g(x)u,y=h(x) \Sigma_o: \quad \begin{cases} \dot{x} &= f(x) + g(x)u, \\ y &= h(x) \end{cases}

Main idea — Select u(x)=ues+udiu(x) = u_{es} + u_{di} that renders the closed-loop system passive.

Σd:{ẋ=fd(x)+g(x)udi,fd:=f(x)+g(x)ues(x)yd=hd(x) \Sigma_d: \quad \begin{cases} \dot{x} &= f_d(x) + g(x) u_{di}, \quad f_d := f(x) + g(x) u_{es}(x) \\ y_d &= h_d(x) \end{cases}

Control problem is cast as a search for HdH_d and hdh_d s.t. Ḣd≤yd⊤udi\dot{H}_d \leq y_d^\top u_{di}

KL-divergence and ELBO

DKL=𝔼θ∼q[logq(θ;z)p(θ∣𝒟)]=logp(𝒟)−𝔼θ∼q[logp(𝒟∣θ)p(θ)q(θ;z)]ℒ(𝒟;z)=𝔼θ∼q[logp(𝒟∣θ)p(θ)−logq(θ;z)] \begin{aligned} D_{\text{KL}} &= \mathbb{E}_{\theta \sim q}\left[ \log \frac{q(\theta; z)}{p(\theta \mid \mathcal{D})}\right] \\ &= \log p(\mathcal{D}) - \mathbb{E}_{\theta \sim q}\left[ \log \frac{p(\mathcal{D} \mid \theta) p(\theta)}{q(\theta; z)} \right] \\ \mathcal{L}(\mathcal{D}; z) &= \mathbb{E}_{\theta \sim q}\left[ \log p(\mathcal{D} \mid \theta) p(\theta) - \log q(\theta; z) \right] \end{aligned}

NeuralPbc

Learning storage function from trajectories

Motivation

Control synthesis in PBC: Gues=∇qH−∇qHd Gu_{es} = \nabla_q H - \nabla_q H_d

NeuralPbc Problem Statement

Consider the mechanical system

[q̇ṗ]=[−01−10][∇qHd∇pHd]+[0G]udi \begin{bmatrix} \dot{q} \\ \dot{p} \end{bmatrix} = \begin{bmatrix} \phantom{-}0 & 1 \\ -1 & 0 \end{bmatrix} \begin{bmatrix} \nabla_q H_{\phantom{d}} \\ \nabla_p H_{\phantom{d}} \end{bmatrix} + \begin{bmatrix} 0 \\ G \end{bmatrix} u_\phantom{di}

Control task: stabilize desired equilibrium x⋆=(q⋆,0)x^\star = (q^\star, 0)

u=ues+udi=G†(∇qH−∇qHd)−KDG⊤∇pHdu = u_{es} + u_{di} = G^{\dagger} \left( \nabla_q H - \nabla_q H_d \right) - K_D G^\top \nabla_p H_d

u=ues+udi=−G†∇qHdθ−KDG⊤∇pHdθu = u_{es} + u_{di} = - G^{\dagger} \nabla_q H_d^{\theta} - K_D G^\top \nabla_p H_d^{\theta}

Choosing a suitable HdH_d is not trivial

Parameterize HdH_d by a neural network HdθH_d^\theta, and relax control task to bringing xx to a small neighborhood of x⋆x^\star

  • Instead of asking for asymptotic stabilization, we only require that trajectories pass thru a nbhd of x⋆x^\star
  • This is reasonable because we know how to stabilize a fixed point, e.g. stabilize the linearization of the system using LQR
  • This allows us to find approximations for HdθH_d^\theta using learning techniques

Set Distance Loss ℓset\ell_{\text{set}}

Penalizes when closed-loop trajectory γ\gamma under the current control law is far away from a neighborhood 𝒮\mathcal{S} of x⋆x^\star

ℓset(x)=inft{∥a−b∥:a∈γ(t),b∈𝒮}\ell_{\text{set}}(x) = \underset{t}{\inf} \left\{ \lVert a-b \rVert : a \in \gamma(t), b \in \mathcal{S}\right\} * The set 𝒮\mathcal{S} may be chosen as * A ball around x⋆x^\star * Estimated region of attraction * No additional loss if any point in γ\gamma is in 𝒮\mathcal{S}

NeuralPbc Loss Function

ℓ≜ℓset(γ)+ℓ⊥(γ,u)\ell \triangleq \ell_{\text{set}}(\gamma) + \ell_{\bot}(\gamma,u)

Transversal Distance Loss ℓ⊥\ell_{\bot}

Measures how close γ\gamma is to γ⋆\gamma^\star (expert trajectory) using transverse coordinates x⊥x_\bot

  • Coordinate transformation
    • τ∈ℝ\tau \in \mathbb{R} a surrogate for time
    • x⊥∈ℝ2n−1x_{\bot} \in \mathbb{R}^{2n-1} quantify how far away the current state is from γ⋆\gamma^\star
